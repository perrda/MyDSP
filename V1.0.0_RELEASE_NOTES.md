## 🔧 **Developer Notes**

### Background Jobs Best Practices

1. **Use appropriate priorities:**
   - `low` - Maintenance, cleanup
   - `normal` - Sync, updates
   - `high` - Reminders, alerts
   - `critical` - Time-sensitive operations

2. **Handle errors gracefully:**
   - Jobs auto-retry 3 times
   - Exponential backoff between retries
   - Failed jobs logged for debugging

3. **Monitor job queue:**
   ```typescript
   import { getJobQueueStats } from './services/backgroundJobs'
   
   const stats = getJobQueueStats()
   console.log(stats) // { total, pending, running, completed, failed }
   ```

### Validation Best Practices

1. **Combine multiple validators:**
   ```typescript
   validationConfig: {
     amount: { 
       rules: [
         validators.positiveAmount(),
         validators.maxAmount(10000)
       ]
     }
   }
   ```

2. **Custom error messages:**
   ```typescript
   validators.ukPostcode('Please enter a valid UK postcode')
   ```

3. **Async validation:**
   ```typescript
   {
     validate: async (value) => {
       const response = await fetch(`/api/validate/${value}`)
       return response.ok
     },
     message: 'Validation failed'
   }
   ```
